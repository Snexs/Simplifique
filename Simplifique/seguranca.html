Segurança

A Segurança do site é feita por padrao https e por encriptografia ssl, todos os seus dados estao seguros conocosco. Nós respeitamos todas as regras da LGPD.

A principal fonte de Segurança que provemos para os usuários é confirmação de dados com a receita federal.

Juntamente com os dados confirmados também temos o sistema de token/validação.

validação Com a Receita Federal

Nós temos o foco no cliente,e para melhor atende-lo e poder prover a maior Segurança possível nós utilizamos dados da receita federal para confirmação e validação dos dados cadastrais.

Tanto prestador quanto contratante vão ter seus dados consultados na base de dados.

Após validado o usuário é liberado para contratar anúncios.

Sistema de Token

Nosso sistema de token é bem simples, após a confirmado o chamado do serviço, e confirmado o pagamento o sistema irá enviar um token para o contratante.

O prestador irá receber o mesmo token e irá confirmar com o do contratante.
